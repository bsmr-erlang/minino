Minino templates
================

You need the minino escript, as well.
``` bash
$ cp ../../bin/minino .
```
Start minino
``` bash
$ ./minino runserver
```

Minino uses [Erlydtl](https://github.com/evanmiller/erlydtl) to render the templates. For more info, please check the [Erlydtl](https://github.com/evanmiller/erlydtl site.
